Node.js + Express + EJS kullanarak tamamen Türkçe bir mobil web uygulaması oluştur.

ÖNEMLİ:
- Tüm arayüz Türkçe olacak.
- Tüm menüler Türkçe olacak.
- Admin paneli Türkçe olacak.
- Sohbet sistemi Türkçe olacak.
- Hata mesajları Türkçe olacak.
- SEO başlıkları ve açıklamaları Türkçe olacak.
- Kod altyapısı Node.js olacak.
- React kullanma.
- Next.js kullanma.
- Vite kullanma.
- Python kullanma.
- PHP kullanma.
- Laravel kullanma.
- Django kullanma.
- Sadece Node.js + Express + EJS kullan.
- Proje Replit üzerinde çalışacak.
- npm install ve npm start ile çalışmalı.

PROJE ADI:
ÖzelGüvenlik.Online tarzında iş ilanları ve topluluk platformu.

AMAÇ:
Türkiye’de özel güvenlik iş ilanlarını, duyuruları ve topluluk sohbetini tek platformda toplayan modern, mobil öncelikli, hızlı ve SEO uyumlu bir web uygulaması yap.

TEKNOLOJİ:
- Node.js
- Express.js
- EJS template engine
- Tailwind CSS veya normal CSS
- Socket.io
- bcrypt
- express-session
- dotenv
- helmet
- express-rate-limit
- SQLite kullan
- Veritabanı kolay taşınabilir olsun

REPLIT UYUMLULUĞU:
Proje şu komutlarla çalışmalı:

npm install
npm start

Port:
process.env.PORT || 3000

package.json içinde şunlar olsun:
"start": "node server.js"
"dev": "nodemon server.js"

CANLI SOHBET:
Socket.io kullan.
Siteye giriş yapmayan kullanıcılar sohbet mesajlarını görebilsin.
Giriş yapmayan kullanıcılar mesaj yazamasın.
Mesaj yazmak isteyince şu uyarı çıksın:
"Mesaj yazmak için giriş yapmalısınız."

Kayıt olan kullanıcılar mesaj yazabilsin.

SOHBET EKRANI:
Üstte şunlar görünsün:
- Aktif kullanıcı sayısı
- Online adminler
- Saat
- Tarih
- Gece/Gündüz hatırlatması

Örnek:
☀️ Günaydın Saat: 08:12
🌙 İyi Geceler Saat: 23:45

SOHBET KURALLARI:
Sohbet ekranında sabit bir kural kutusu olsun.

Metin:
📢 Sohbet Kuralları
- Hakaret yasaktır.
- Küfür yasaktır.
- Cinsel içerik yasaktır.
- Porno içerik yasaktır.
- Irkçılık yasaktır.
- Reklam yapmak yasaktır.
- Spam yapmak yasaktır.
- Kuralları ihlal eden kullanıcılar geçici veya kalıcı olarak yasaklanabilir.

Bu kural kutusu her zaman görünür olsun.

TÜM SAYFALARDA SOHBET BALONU:
